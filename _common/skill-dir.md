## Skill Directory

This skill includes bundled scripts and tools. Reference them using `SKILL_DIR`:

```bash
# SKILL_DIR points to this skill's installation directory
# Use it to reference bundled scripts, binaries, and templates
```

**Examples:**
- `$SKILL_DIR/bin/rebyte.js` - bundled CLI
- `$SKILL_DIR/scripts/init.sh` - setup script
- `$SKILL_DIR/templates/` - template files
