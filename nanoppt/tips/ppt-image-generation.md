## Text Rendering Tips

### Use Quotes for Exact Text

```
❌ Write the title
✅ Center large title text "Skills" in bold white font
```

### Specify Position

```
❌ Text in image
✅ Top center / bottom right corner / left side
```

### Specify Style

```
❌ Big title
✅ Bold white font, 48px / smaller gray text
```

## Prompt Templates

### Cover Page

```
[Style] presentation cover slide, clean minimalist,
[Background description].
Center large title text "[Title]" in bold white font,
subtitle below "[Subtitle]" in smaller text,
bottom right small text "[Author]",
16:9 ratio
```

### Section Divider

```
[Style] section divider slide,
[Background color] gradient background,
large centered text "[Number]" and "[Section Title]",
clean design, no other elements, 16:9 ratio
```

### Content Page

```
[Style] illustration for presentation,
[Theme description],
[Color scheme],
no text, clean background, 16:9 ratio
```
